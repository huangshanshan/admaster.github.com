TrackMaster™ API - 首页

此 API 接口是 TrackMaster™ 提供给第三方开发者的编程接口。您可以通过 API 操作有关广告主、广告位、点位等项目信息。

在开发之前你需要在管理应用页面申请创建一个应用，填入应用的描述信息，从而获得 Client ID 和 Client Secret，这个 Client ID 用于唯一标识你的应用。创建完应用以后，使用 OAuth2.0 对用户进行验证，保障用户的隐私和安全性。

使用 TrackMaster™ API 进行应用开发，请先阅读开发者协议。

工作网络

API 说明 参数
GET /networks 获取当前授权用户有权操作的所有工作网络 0
GET /networks/:id 获取指定 ID 工作网络信息 1

广告主

API 说明 参数
GET /advertisers 获取系统广告主列表 0
GET /advertisers/:id 获取指定 ID 系统广告主信息 1
GET /networks/:network_id/advertisers 获取指定 ID 工作网络下所有广告主信息 1
GET /networks/:network_id/advertisers/:id 获取指定 ID 网络下的指定 ID 广告主信息 2
PUT /networks/:network_id/advertisers/:id 添加指定 ID 系统广告主到指定 ID 工作网络下 2
PATCH /networks/:network_id/advertisers/:id 修改指定 ID 网络下的指定 ID 广告主 2
DELETE /networks/:network_id/advertisers/:id 删除指定 ID 工作网络下的指定 ID 广告主 2

品牌

API 说明 参数
GET /networks/:network_id/advertisers/:advertiser_id/brands 获取指定 ID 网络下指定 ID 广告主下的所有品牌 2
GET /networks/advertisers/brands/:id 获取指定 ID 品牌详细信息 1
POST /networks/:network_id/advertisers/:advertiser_id/brands 添加指定 ID 品牌到指定 ID 网络广告主下 2
PATCH /networks/advertisers/brands/:id 修改特定 ID 的网络广告主下品牌名称 1
DELETE /networks/advertisers/brands/:id 删除指定 ID 的网络广告主下品牌 1

媒体

API 说明 参数
GET /medias 获取系统媒体库列表 0
GET /medias/:id 获取指定系统媒体详细信息 1
GET /networks/:network_id/medias 获取指定工作网络下媒体库列表 1
GET /networks/medias/:id 获取指定工作网络下指定媒体信息 1
POST /networks/:network_id/medias 给指定工作网络添加一个媒体 1
DELETE /networks/medias/:id 删除指定工作网络下的媒体 1
PATCH /networks/medias/:id 修改指定工作网络下指定媒体属性 1
GET /networks/advertisers/campaigns/:campaign_id/medias 获取指定项目下已添加的媒体 1
GET /networks/advertisers/campaigns/:campaign_id/medias/:id 获取指定项目下指定媒体详细信息 2
PUT /networks/advertisers/campaigns/:campaign_id/medias/:id 为指定项目添加指定媒体 2
DELETE /networks/advertisers/campaigns/:campaign_id/medias/:id 删除指定项目下某个媒体 2

项目

API 说明 参数
GET /networks/:network_id/advertisers/:advertiser_id/campaigns 获取指定网络下某个广告主有操作权限的项目列表 2
GET /networks/advertisers/campaigns/:id 获取指定项目信息 1
POST /networks/:network_id/advertisers/:advertiser_id/campaigns 添加指定项目到指定广告主下 2
DELETE /networks/advertisers/campaigns/:id 删除指定项目 1
PATCH /networks/advertisers/campaigns/:id 修改指定项目属性 1

广告位

API 说明 参数

GET /networks/advertisers/campaigns/placements/:id 获取指定广告位信息 1
POST /networks/advertisers/campaigns/:campaign_id/placements 添加一个广告位在指定项目下 1
DELETE /networks/advertisers/campaigns/placements/:id 删除指定的广告位 1
PATCH /networks/advertisers/campaigns/placements/:id 修改指定的广告位属性 1

GET /networks/advertisers/campaigns/creatives/:id 获取项目下指定创意信息 1
POST /networks/advertisers/campaigns/:campaign_id/creatives 在指定项目下添加创意 1
DELETE /networks/advertisers/campaigns/creatives/:id 删除指定的创意 1
