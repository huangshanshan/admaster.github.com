品牌

  • 获取指定网络下指定广告主下的所有品牌
  • 获取指定 ID 品牌详细信息
  • 添加指定品牌到指定网络广告主下
  • 修改指定的网络广告主下品牌名称
  • 删除指定的网络广告主下品牌

获取指定网络下指定广告主下的所有品牌

GET /networks/:network_id/advertisers/:advertiser_id/brands

响应

Status: 200 OK
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999

[
  {
    "id": 1,
    "name": "巧乐兹",
    "created_at": "2012-09-06T20:39:23Z"
  }
]

获取指定 ID 品牌详细信息

GET /networks/advertisers/brands/:id

响应

Status: 200 OK
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999

{
    "id": 1,
    "network_id": 1,
    "advertiser_id": 10231,
    "name": "巧乐兹",
    "url": "http://track.admasterapi.com/networks/advertisers/brands/1",
    "created_at": "2012-09-06T20:39:23Z"
}

添加指定品牌到指定网络广告主下

POST /networks/:network_id/advertisers/:advertiser_id/brands

请求

{
    "name": "巧乐兹"
}
name
必填 string - 品牌名称

响应

Status: 201 Created 
Location: http://track.admasterapi.com/networks/1/advertisers/10231/brands
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999

{
    "id": 1,
    "network_id": 1,
    "advertiser_id": 10231,
    "name": "巧乐兹",
    "url": "http://track.admasterapi.com/networks/advertisers/brands/1",
    "created_at": "2012-09-06T20:39:23Z"
}

修改指定的网络广告主下品牌名称

PATCH /networks/advertisers/brands/:id

请求

{
    "name": "巧乐鸡"
}
name
必选 string - 品牌名称

响应

Status: 204 No Content 
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999

删除指定的网络广告主下品牌

DELETE /networks/advertisers/brands/:id

响应

Status: 204 No Content 
Location: http://track.admasterapi.com/networks/1/advertisers/10231/brands
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999
